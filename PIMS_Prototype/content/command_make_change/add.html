Official Git Documentation - Add

Add (AKA 'stage')

DETAILS

Performing an 'add' / staging' is the process of updating (but not comitting) the current branch with new or modified files. This command can be performed multiple times before a commit. It only adds the content of the specified file(s) at the time the add command is run.

git add .

Adds all files / folders that or have changed. Note: will only add files/folders in the current folder / sub-folder.

git add *.html

Adds changed files that have an extension of 'html'.
