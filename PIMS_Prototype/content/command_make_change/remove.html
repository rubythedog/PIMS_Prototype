Official Git Documentation - Remove

Remove (AKA 'untrack')

DETAILS

Performing a 'remove' is the process of removing a file / folder from version control (), and leaving it in your working folder.

git rm -f --cached -- [PATHSPEC]

Removes from being tracked, but does not delete it.

-f force removal

--cached do not delete file after it is unstaged

git rm -f --cached --*.html

Removes 'staged' files wtih an extension of 'html'.
