About James' Git Tutorial

Created by me (James Fowler) to teach myself Git.

This website is stored in GitHub, thereby forcing me to learn about Git to store the website artifacts.

This technique (forcing a tutorial about Git to be stored in Git repository GitHub) is a variant on dogfooding.

This web-site was created using the following technologies:

  • HTML (of course)
  • JavaScript (of course)
  • jQuery
  • BootStrap (NavBar)
  • Font-Awesome 4.70
  • Google Fonts (Droid Sans Mono, Noto Serif, Open Sans)
  • CSS Styles copied from the Git Manual Page
