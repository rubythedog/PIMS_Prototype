Restore (AKA 'undo changes')

DETAILS

Performing an 'restore' is the process of updating the with the contents of the remote source.

Note: 'Restoring' a deleted file / folder is the same as un-deleting it.

git checkout -- [PATHSPEC]

'Un-does' the change, or 'un-deletes' .

git checkout -- *.html

'Un-does' changes, or 'un-deletes' files with an extension of 'html'.
