Official Git Documentation - Clean

Clean

DETAILS

Performing a 'clean' results in all the untracked files/folders from the being deleted.

git clean -d -f

Deletes any untracked files/folders.

-d deletes files/folders in sub-folders

-f forces the removal if the git configuration file requires the force flag.

git clean -d -f --dry-run

Deletes any untracked files/folders.

-d deletes files/folders in sub-folders

-f forces the removal if the git configuration file requires the force flag.

--dry-run don’t actually remove anything, just show what would be done.
