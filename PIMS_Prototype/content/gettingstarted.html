Getting Started

Before anything can be done using Git, client software must be installed. It's also recommended that the reference documents be reviewed.

1. Install the following software

Git for Windows (required)

Microsoft Visual Studio Team Explorer 2019 (recommended)

Microsoft Visual Studio Code (recommended)

SourceTree Git Client (optional, but recommended)
