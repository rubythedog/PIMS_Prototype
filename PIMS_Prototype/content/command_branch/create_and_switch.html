Official Git Documentation - Switching Branches

Create and Switch

DETAILS

Allows a branch to be created (using an existing branch as a 'base'), then immediately switches to it to allow work to be done.

git checkout -b <new_branch> <base_branch>

Note: <base_branch> must exist locally.

-b Creates a new branch named <new_branch>

This is the transactional equivalent of:

$ git branch -f <new_branch> [<base_branch>]  (1)
$ git checkout <new_branch>                    (2)

  1. create branch from another branch

  2. switch to newly created branch
