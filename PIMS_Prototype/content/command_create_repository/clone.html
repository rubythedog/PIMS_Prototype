Official Git Documentation - Clone

Clone

DETAILS

Clone (downloads) a repository, including all of the files, branches, and commits.

git clone [url]

Clones (downloads) a repository into a newly created directory. The newly created directory will be the last 'folder' in the URL.

The URL is obtained from the Git Repository (i.e. TFS, GitHub, etc.).

EXAMPLE

If the current folder is "c:\data\GIT". The following command: 

git clone https://github.com/SampleProject/SampleRepo

will result in a repository being created located at "c:\data\GIT\SampleRepo"
