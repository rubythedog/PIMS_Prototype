View (file) History

If you want to see what changes were made for a file, the best way (I've found), is to use Visual Studio (not code) Team Explorer.

Steps

Prerequisite: Install Microsoft Visual Studio Team Explorer

  1. Launch Visual Studio 2017/2019.
  2. Click File|Open|Folder...
  3. Select the 'root' folder of the local Git Repository.
  4. In Solution Explorer, find the file to view the history of
  5. Right click on the file, select View History...
